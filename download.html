Installation guide

  1. Open the installer

    Double-click Threde-Installer.pkg. If macOS shows a security prompt, open System Settings → Privacy & Security and click Open Anyway. This appears because the app is new — it is fully notarized by Apple.

  2. Get a free Gemini API key

    Go to aistudio.google.com/app/apikey, sign in with any Google account, and click Create API key. Copy the key — you'll paste it into Threde in the next step.

  3. Complete the setup screen

    After installation, a setup page opens in your browser. Paste your Gemini API key and click Continue. Then click Sign in with Google to grant read-only Gmail access.

  4. Your first briefing

    Threde runs automatically every morning at 5 AM. To run it immediately, open Terminal and type: open http://localhost:5050 — if Threde has not yet generated a briefing it will do so on first visit.

Before you start: you need a Google account (Gmail), an internet connection for the initial setup and daily fetch, and macOS 13 Ventura or later. Threde stores everything — your API key, Gmail token, and all summaries — only on your computer. See the privacy policy for details.
